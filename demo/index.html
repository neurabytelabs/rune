How It Works

Phase 1 — Input

📝 Your Prompt

You write one prompt. That's it. SWARM handles the rest.

Phase 2 — Spawn

🐝 Agent Swarm Deploys

3-6 agents spawn in parallel, each with a radically different thinking strategy: Expert, Creative, Devil's Advocate, Synthesizer, Minimalist, Wild Card.

Phase 3 — Compete

⚔️ Spinoza Tournament

Every response is scored on 4 philosophical pillars: Conatus (action), Ratio (logic), Laetitia (tone), Natura (flow). Only the strongest survive.

Phase 4 — Evolve

🧬 Cross-Pollination

In Round 2, agents see the top performers' outputs. They evolve. Scores improve. Darwinian prompt evolution in action.

Phase 5 — Fuse

🔮 Synthesis

Top agents' best insights merge into a single, definitive response. Better than any individual agent could produce alone.
